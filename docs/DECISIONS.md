# Decisions

- Project name: MCP Agent Discovery Dashboard
- Repository name: mcp-agent-discovery-dashboard
- Frontend will be React.
- Frontend will be hosted on Cloudflare Pages.
- Backend will be Node.js.
- Backend will be hosted on Render.
- Repository will be on GitHub.
- Backend will contain MCP client logic.
- React frontend will not directly call MCP servers.
- LLM and API keys will stay only in backend environment variables.
- First version will use demo MCP tools before adding real external MCP servers.
- Agent graph will show each step of the MCP flow.
